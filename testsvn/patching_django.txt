In order to use a patched copy of Django, complete the following steps.

1. Make a global Django dir somewhere, i.e. c:\django.
2. Create subdir 'trunk' under c:\django.
3. From c:\django\trunk\ execute:

    svn checkout http://code.djangoproject.com/svn/django/trunk/django django
    svn checkout http://code.djangoproject.com/svn/django/tests tests
  
  These commands obtain the latest Django source and test code.

4. From the c:\python24\Lib\site_packages delete the 'django' subdir and all files underneath.
5. In the same subdir, create a new file, django.pth. Populate the file with:

    # Trunk is a svn checkout of:
    #   http://code.djangoproject.com/svn/django/trunk/
    #
    c:/django/trunk/
    
    # Patched versions:
    #
    #c:/django/5268-p3297/
